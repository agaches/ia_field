# Framework IA pour l'Employé

> Adoption IA individuelle pour améliorer votre productivité personnelle

## Pour qui ?

Ce framework s'adresse aux **individus** (tech et non-tech) souhaitant utiliser l'IA pour leur productivité :
- Développeurs (GitHub Copilot, Cursor, etc.)
- Employés bureautiques (ChatGPT, Claude, M365 Copilot)
- Professionnels créatifs (writing, research, analyse)

**Maturité** : Niveaux 1-2 (basique à intermédiaire) - Voir [GLOSSARY.md](../GLOSSARY.md)

## Les 6 phases

1. **[Strategy](01-strategy.md)** - Découvrir les outils IA disponibles
2. **[Plan](02-plan.md)** - Évaluer et développer vos compétences
3. **[Ready](03-ready.md)** - Configurer votre environnement
4. **[Govern](04-govern.md)** - Comprendre les règles d'usage
5. **[Secure](05-secure.md)** - Adopter les bonnes pratiques sécurité
6. **[Manage](06-manage.md)** - Gérer votre productivité

## Quick Start

1. Lisez [Strategy](01-strategy.md) pour découvrir les outils
2. Testez 2-3 outils pendant 1 mois
3. Consultez [Secure](05-secure.md) pour les règles de base
4. Évaluez vos résultats

**Timeline** : Quelques heures à quelques jours
**Coût** : 20-50€/mois
