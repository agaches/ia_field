## 2. Detailed Level Definitions

### Level 1: BASIC

**Profile**: Individual discovering AI tools for personal productivity

**Capabilities**:
- Use conversational AI (ChatGPT, Claude, Gemini) for writing, research, brainstorming
- Use code assistants (GitHub Copilot) for basic code completion
- Write effective prompts with clear context and instructions
- Understand AI limitations (hallucinations, bias, accuracy issues)
- Follow data classification rules (know what NOT to share)

**Tools mastered**:
- 1-2 conversational AI tools (SaaS)
- 1 code assistant OR 1 productivity tool
- Free or basic paid tiers ($0-20/month)

**Time investment**: 5-10 hours learning + daily practice

**Validation**:
- Successfully automate 2-3 daily tasks with AI
- Write 3 effective prompts demonstrating context awareness
- Pass basic AI concepts quiz (80%)

---

### Level 2: INTERMEDIATE

**Profile**: Proficient AI user integrating tools into daily workflow

**Capabilities**:
- Advanced prompt engineering (few-shot, chain-of-thought, role-based)
- Use multiple AI tools in combination for complex tasks
- Basic API usage (OpenAI, Anthropic) for simple automations
- Understand tokens, costs, and context windows
- Share AI best practices with team members
- Validate AI outputs critically and systematically

**Tools mastered**:
- 3-4 AI tools across categories (conversational, code, media, productivity)
- API basics (make calls, handle responses, manage tokens)
- Integration with personal workflow tools

**Time investment**: 20-40 hours learning + 3-6 months practice

**Validation**:
- Complete 1 API-based automation project
- Document 5 reusable prompt templates
- Achieve 2+ hours/week productivity gains (self-reported)

---

### Level 3: ADVANCED

**Profile**: Technical specialist building custom AI solutions

**Capabilities**:
- Build RAG (Retrieval-Augmented Generation) applications
- Work with vector databases and embeddings
- Implement basic MLOps (versioning, testing, monitoring)
- Use frameworks (LangChain, LlamaIndex) for complex workflows
- Design and evaluate prompts systematically
- Understand LLM architectures and trade-offs (GPT vs Claude vs Llama)
- Lead team AI adoption initiatives

**Tools mastered**:
- RAG frameworks (LangChain, LlamaIndex)
- Vector databases (Pinecone, Weaviate, Chroma)
- Cloud AI platforms (SageMaker, Vertex AI, Azure ML - basics)
- Basic MLOps tools (Git, DVC, simple CI/CD)

**Time investment**: 60-100 hours learning + 6-12 months practice

**Validation**:
- Build functional RAG application (<2s latency p95)
- Deploy team-level AI integration (code review, testing, docs)
- Write technical documentation for AI systems
- Pass advanced AI/ML certification (AWS/GCP/Azure)

---

### Level 4: EXPERT

**Profile**: AI architect deploying production systems at scale

**Capabilities**:
- Fine-tune LLMs (LoRA, QLoRA, RLHF) on custom datasets
- Design multi-region, high-availability AI infrastructure
- Implement complete MLOps/LLMOps pipelines (CI/CD/CT)
- Optimize GPU usage and costs at scale
- Conduct responsible AI audits (NIST AI RMF)
- Lead organizational AI strategy and governance
- Mentor teams on AI best practices

**Tools mastered**:
- Production infrastructure (Kubernetes, Terraform, GPU clusters)
- Advanced ML frameworks (DeepSpeed, FSDP, Ray)
- Complete MLOps stack (MLflow, Kubeflow, W&B)
- Monitoring and observability (Prometheus, Grafana, LangSmith)
- Multiple cloud providers (AWS/GCP/Azure)

**Time investment**: 200+ hours learning + 12-24 months practice

**Validation**:
- Deploy production AI system with SLA (99.9% uptime, <2s p95 latency)
- Complete LLM fine-tuning project with measurable ROI
- Pass expert-level certifications (AWS ML Specialty, CKAD)
- Lead organizational AI Center of Excellence

## 4. Self-Assessment Guide

### How to Assess Your Level

**Step 1: Domain Assessment**
- Rate yourself 1-4 in each domain (A-G above)
- Be honest - overestimating hurts your progression

**Step 2: Calculate Average**
- Sum all domain scores
- Divide by 7 (number of domains)
- Round to nearest level

**Step 3: Validation Check**
- Can you demonstrate the "Validation" criteria for your level?
- If not, adjust down until you meet all validation criteria

**Step 4: Choose Your Framework**
- Level 1-2 → Start with [Employee Framework](employee/README.md)
- Level 2-3 → Progress to [Delivery Framework](delivery/README.md)
- Level 3-4 → Advance to [Automation Framework](automation/README.md)

### Example Self-Assessment

**Developer Profile**:
- Conversational AI: 2 (use Claude daily with advanced prompts)
- Code Assistance: 2 (Copilot + Cursor, generate tests)
- AI/ML Fundamentals: 1 (understand basics, no API experience)
- AI Tooling: 1 (SaaS only)
- Data & Security: 2 (follow team policies, anonymize data)
- Governance & Ethics: 1 (validate outputs, acknowledge use)
- Operations: 1 (personal subscriptions)

**Average**: (2+2+1+1+2+1+1) / 7 = 1.4 → **Level 1-2**
**Framework**: Employee → Delivery (transition phase)
